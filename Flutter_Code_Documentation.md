## Main App Structure

### `lib/main.dart`
**Entry point of the Flutter application**
- Initializes Firebase and sets up the app with Riverpod state management
- Contains AuthWrapper that handles user authentication state and role-based navigation
- Routes users to appropriate screens based on their role (customer, center admin, super admin)

## Summary

This Flutter application, "Flow and Glow," is a comprehensive wellness platform that connects customers with yoga, pilates, and meditation centers. The app features role-based access control for customers, center administrators, and super administrators, with a modern Material Design 3 interface using a warm, earthy color scheme. The architecture follows clean code principles with proper separation of concerns, utilizing Riverpod for state management, Firebase for authentication and data storage, and providing a complete booking and subscription management system.
